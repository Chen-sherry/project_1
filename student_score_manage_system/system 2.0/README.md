学生成绩管理系统 2.0（GUI 图形化版）

一、项目简介

&#x09;基于 Python + Tkinter + MySQL 开发的图形化学生成绩管理系统，在 1.1 命令行版本基础上全面升级为可视化界面，操作更直观、交互更友好。保留原有数据管理、权限校验、操作日志等核心能力，新增窗口生命周期管理、异常防护、输入规范化校验、界面自动布局等企业级特性，适合作为课程设计、实训项目与小型管理系统实战使用。

二、技术栈

&#x09;开发语言：Python 3

&#x09;界面框架：Tkinter（原生 GUI 图形化）

&#x09;数据库：MySQL

&#x09;数据库驱动：PyMySQL

&#x09;依赖库：cryptography、socket、time、tkinter.messagebox、tkinter.ttk

三、核心功能（完整继承 1.1 所有能力）

&#x09;学生信息管理

&#x09;	添加、修改、查询、删除学生；删除时自动校验关联成绩，支持级联清理。

&#x09;教师信息管理

&#x09;	添加、修改、查询、逻辑删除教师；状态字段自动校验，保留历史数据。

&#x09;学生成绩管理

&#x09;	成绩录入、多维度查询、按考试次数修改；成绩范围自动限制 0\~100。

&#x09;权限与日志

&#x09;	数据库账号登录验证；操作日志自动记录（操作人、时间、IP、内容）。

&#x09;登录安全

&#x09;	最多 5 次登录重试，次数耗尽自动退出。

四、1.1 → 2.0 完整升级说明

&#x09;1. 界面全面升级：命令行 → 图形化 GUI

&#x09;	从纯黑框操作升级为 窗口化图形界面

&#x09;	主界面功能菜单按钮化，点击即用

&#x09;	所有输入改为输入框、单选框，不再手动输入命令

&#x09;2. 交互体验大幅优化

&#x09;	所有操作弹窗提示（成功 / 失败 / 错误 / 确认）

&#x09;	查询结果自动表格展示，清晰可读

&#x09;	子窗口模态化，防止误点主界面

&#x09;3. 输入校验强化（企业级规范）

&#x09;	空值自动拦截

&#x09;	年龄、成绩、ID 强制数字校验

&#x09;	成绩范围 0\~100、年龄 1\~99 严格限制

&#x09;	所有校验失败 立即终止执行，不继续跑数据库逻辑

&#x09;4. 窗口与资源管理升级

&#x09;	主窗口关闭自动写 登出日志

&#x09;	关闭窗口自动释放数据库连接，不泄露连接

&#x09;	子窗口关闭自动释放焦点锁，程序不卡死

&#x09;	支持右上角 × 正常退出，不再强制崩溃

&#x09;5. 稳定性与健壮性提升

&#x09;	所有数据库操作增加事务回滚

&#x09;	异常捕获更全面，提示更明确

&#x09;	按钮防重复点击、界面自动适配

&#x09;	修复大量逻辑错误（字段拼写、条件错误、返回值不统一）

&#x09;6. 操作流程更规范

&#x09;	登录 → 主菜单 → 功能子窗口

&#x09;	修改 / 查询支持 “先查看再编辑”，防止误操作

&#x09;	所有功能独立窗口，互不干扰

六、版本对比总结

&#x09;1.1：命令行、纯文本、适合学习基础逻辑

&#x09;2.0：GUI 图形化、企业级规范、健壮稳定、可交付演示
