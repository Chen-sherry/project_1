学生成绩管理系统 1.1

========================================

一、项目简介

&#x09;基于 Python + MySQL 开发的轻量级命令行学生成绩管理系统，实现学生、教师、成绩的增删改查核心功能，代码结构清晰、操作简洁，适合 Python 入门实战的基础项目，支持多用户权限控制与操作日志追溯。

二、技术栈

&#x09;开发语言：Python 3

&#x09;数据库：MySQL

&#x09;数据库驱动：pymysql

&#x09;辅助依赖：cryptography（加密支持）、socket（IP 获取）、time（时间记录）、os（系统命令）

三、核心功能

&#x09;学生信息管理

&#x09;	支持学生信息添加（姓名、性别、年龄）、修改、多方式查询（全体查询 / 单 ID 查询）

&#x09;	支持学生删除（需管理员权限），删除时自动校验关联成绩记录，避免数据残留

&#x09;教师信息管理

&#x09;	支持教师信息添加（姓名、授课科目）、修改、多方式查询（全体查询 / 单 ID 查询）

&#x09;	支持教师逻辑删除（仅更新状态字段，保留历史数据），需管理员权限

&#x09;成绩管理

&#x09;	支持成绩录入（关联学生 ID、考试次数、改卷教师 ID），成绩范围校验（0-100 分）

&#x09;	支持成绩查询（按学生姓名 / ID 查询，显示考试次数、各科成绩、改卷教师）

&#x09;	支持成绩修改（按学生 ID + 考试次数精准修改，修改后实时展示更新结果）

&#x09;权限与日志

&#x09;	区分管理员与普通用户，普通用户无删除操作权限

&#x09;	操作日志自动记录（操作人、操作类型、操作时间、操作 IP），便于追溯

&#x09;优化删除逻辑：

五、环境准备

&#x09;安装 Python 3.x（推荐 3.6 及以上版本）

&#x09;安装依赖包：

&#x09;	pip install pymysql cryptography

六、运行方式

&#x09;命令行执行：python stu\_sco\_system\_admin(1.1).py，按菜单提示完成操作即可

七、打包为可执行文件

&#x09;安装打包工具：

&#x09;	pip install pyinstaller

&#x09;执行打包命令（Windows 环境）：

&#x09;	python -m PyInstaller -F -c --hidden-import=pymysql --hidden-import=cryptography stu\_sco\_system\_admin(1.1).py

&#x09;	打包完成后，可执行文件位于程序目录下的 dist 文件夹中，直接双击即可运行
