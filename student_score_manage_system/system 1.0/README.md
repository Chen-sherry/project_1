学生成绩管理系统

项目简介

&#x09;基于 Python+MySQL 开发的轻量级命令行学生成绩管理系统，实现学生、教师、成绩的增删改查核心管理功能，适合 Python 入门实战的基础项目。

技术栈

&#x09;开发语言：Python 3

&#x09;数据库：MySQL

&#x09;数据库驱动：pymysql

核心功能

&#x09;学生信息：添加、删除、修改、查询

&#x09;教师信息：添加、删除、修改、查询

&#x09;成绩管理：录入、查询、修改，支持关联考试次数和改卷教师

环境准备

&#x09;安装 Python 3 环境

运行方式

&#x09;终端进入项目文件目录

&#x09;执行命令：python stu\_sco\_system\_admin.py

&#x09;按终端菜单提示输入序号，执行对应功能
